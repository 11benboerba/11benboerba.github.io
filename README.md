# 楚了个楚 - 一个消除游戏 🎮

## 游戏介绍 🌟

《楚了个楚》是一个非常无聊的消除游戏，虽然玩法无聊简单，但饱含了我们满满的情谊！饱含了赵大天才对大家满满的爱！总之！这是一个用来纪念我们友情的小游戏，希望能带给大家欢乐！爱大家❤️💋

## 游戏玩法 🎯

1. 点击上方卡牌将其移至下方
2. 收集相同的卡牌
3. 凑齐三张相同卡牌即可消除
4. 成功消除所有卡牌即可获胜！

## 特别彩蛋 ✨

游戏中包含了许多有趣的彩蛋：
- 消除赵大天才：显示"赵大天才真帅"
- 消除超级鸡：显示"敌军超级鸡已被消灭"
- 消除张几样：显示"张几样已撤退"
- 更多彩蛋等你发现...
